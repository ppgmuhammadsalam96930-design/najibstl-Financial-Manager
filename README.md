## Keahlian teknis
- **Frontend:** HTML, CSS, JavaScript — fokus pada UI kaya & pengalaman pengguna (mobile-first).  
- **Backend:** Python (Flask) — API ringan, otentikasi JWT, integrasi MongoDB.  
- **Database:** MongoDB Atlas (cloud) & setup lokal via Docker.  
- **Dev tooling:** Docker, Docker Compose, GitHub Actions, Nix (opsional).  
- **Keamanan & praktik baik:** `.env` handling, CSP headers, rate-limiting, server-side auth.

  - Fitur: offline UI, server-side protected Mongo bridge, CLI `stl`, Docker + Nix support, GitHub Actions CI.
- Kontribusi dan eksperimen lain (tools lokal & patch UI) — tersedia di repo.
